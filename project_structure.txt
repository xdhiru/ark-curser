# Ark Curser - Automation Bot

## Project Structure

ark-curser/
│
├── main.py                 # Main application entry point - initializes ark_curser and orchestrates tasks
├── screenshot.py           # Utility script for manual screenshot capture and debugging
├── requirements.txt        # Python dependencies (easyocr, pyyaml, opencv-python, numpy, etc)
├── README.md               # Project documentation and setup instructions
│
├── utils/                  # Core utility modules
│   ├── adb.py              # ADB device communication (tap, swipe, screenshot, device status)
│   ├── vision.py           # Template matching and image detection using OpenCV
│   ├── ocr.py              # OCR operations for timer scanning and text extraction (EasyOCR)
│   ├── logger.py           # Centralized logging system (console INFO+, file DEBUG+)
│   ├── time_helper.py      # IST time formatting, remaining time calculation, and scheduling
│   ├── config_loader.py    # YAML configuration loader and validator
│   └── __init__.py         # Package initialization
│
├── tasks/                  # Automation task modules
│   ├── navigation.py       # Screen navigation (reach_home, reach_base, find_trading_posts)
│   ├── handle_trading_posts.py  # TradingPost class and cursing protocol orchestration
│   └── __init__.py         # Package initialization
│
├── logs/
│   └── ark_curser.log
│    
└── config/                 # Configuration management
    ├── settings.yaml       # Main configuration (screen coordinates, paths, GPU settings, timeouts)
    └── user_templates/     # User-customizable template images (overrides core templates)
